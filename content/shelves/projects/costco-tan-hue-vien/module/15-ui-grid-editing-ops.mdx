## Yêu cầu hiển thị
- Grid tương tự Excel, cuộn mượt, hỗ trợ hàng trăm–nghìn nhân viên.
- Freeze cột trái: mã NV, họ tên, phòng ban, ca, ngày công, phút trễ, giờ tăng ca.
- Cột ngày hiển thị ký hiệu + giờ vào/ra (2 dòng).

## Thao tác hiệu chỉnh
- Chỉnh sửa trực tiếp ký hiệu/giờ ngay trên grid.
- Hỗ trợ undo/redo ≥ 30 bước (spec).
- Audit log thay đổi (ai/khi nào/giá trị cũ-mới).

## Evidence từ code (Odoo UI)
- View form/tree trong `hr_monthly_attendance_distribution_view.xml` hiển thị `day_1..day_31`, `actual_workdays`, `late_minutes`, `overtime_hours`.
- Widget `colored_cell_widget.js` tô màu theo giá trị (X, A, H) cho ô ký hiệu.

## Observation / Inference / Decision
- **Observation**: UI code hiện dùng tree editable trong Odoo, chưa phản ánh toàn bộ UX yêu cầu giống Excel.
- **Inference**: Cần xác định gap giữa yêu cầu UI và khả năng hiển thị hiện tại.
- **Decision**: Note này mô tả yêu cầu + evidence UI hiện hữu để phục vụ roadmap.
