## Spec rule
- Nếu có OT thì ≥ 60 phút/ngày.
- OT giữa hai ngày nghỉ ≤ 12 giờ.
- OT đồng đều trong phòng ban (chênh ≤ 30 phút/ngày).

## Code evidence
- `distribute_overtime_hours`: phân bổ OT ngẫu nhiên 1–4 giờ/ngày từ ngày 15 trở đi.
- `process_attendance_distribution`: OT tăng checkout_time theo giờ chuẩn + OT + random_minutes.

## Drift/Gap
- Code không thể hiện rõ điều kiện OT giữa hai ngày nghỉ hoặc cân bằng OT trong phòng ban.
- Ngưỡng OT theo ngày trong code dùng 1–4 giờ, không trực tiếp ràng buộc 60 phút.

## Observation / Inference / Decision
- **Observation**: Phân bổ OT trong code thiên về random có kiểm soát.
- **Inference**: Rule OT spec chưa được phản ánh đầy đủ trong code.
- **Decision**: Đưa vào traceability matrix + conflict note để xác định mức độ drift.
