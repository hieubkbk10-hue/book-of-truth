## Observation / Inference / Decision
- **Observation**: Nhiều rule có drift/unknown, nhất là đi trễ, OT và cân bằng ca.
- **Inference**: Cần xác nhận spec hay code là nguồn chuẩn cho từng rule.
- **Decision**: Matrix là nguồn tham chiếu chính cho note conflict.
